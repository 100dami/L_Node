# About __

## 정적언어 vs 동적언어 
<br>

### 정적언어
> 자료형을 선언할 때 지정해 주는 언어    
  대표적으로는 C, C++, C#, Java 
```c
int num = 1; // 컴파일 성공
float num2 = 1.03; // 컴파일 성공
bool num3 = 1; // 컴파일 에러
```    
### 동적언어
> 자료형을 실행할 떼 결정하는 언어
  대표적으로는 JavaScript, Ruby, Python, SmallTalk     
``` python
num1 = 10 # number
name = "Evan Hwang" #String
```
